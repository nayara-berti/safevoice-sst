## 🇧🇷 Português

### 💙 Sobre o Repositório
Este repositório documenta a evolução do projeto **SafeVoice SST**.

Ele apresenta a construção de uma solução voltada para **orientação preliminar em Segurança do Trabalho**, utilizando interação por voz, transcrição automática, consulta a uma base local de conhecimento e resposta em áudio.

O projeto foi desenvolvido com foco em:
- acessibilidade;
- consulta preliminar em SST;
- organização modular;
- aplicação de conceitos de **Engenharia de Software** e **Orientação a Objetos**.

### 🎓 Contexto do Projeto
O **SafeVoice SST** foi desenvolvido a partir de um desafio prático da trilha **GenAI & Dados**, no contexto do bootcamp promovido pelo **Bradesco** em parceria com a **DIO.me**.

A proposta inicial foi expandida com uma abordagem mais estruturada, conectando **transcrição de voz, síntese de fala, organização arquitetural e aplicação prática em Segurança do Trabalho**.

### 📂 notebooks/
Versão funcional inicial desenvolvida no **Google Colab**.

- gravação de áudio no navegador  
- transcrição de fala com **Whisper**  
- consulta a uma base local por palavras-chave  
- resposta orientativa em texto  
- conversão da resposta em áudio com **gTTS**  

### 📂 estrutura lógica atual
Versão com organização inicial inspirada em princípios de arquitetura.

- separação de responsabilidades  
- uso de classes para serviços principais  
- divisão entre transcrição, base de conhecimento, síntese de voz e controle do fluxo  
- estrutura preparada para futura modularização em arquivos Python  

### 🎯 Objetivo
Demonstrar evolução técnica na construção de uma solução aplicada à Segurança do Trabalho:

**Protótipo por voz → Arquitetura estruturada → Base para assistente inteligente em SST**

### 🤖 Próxima Etapa
A próxima evolução do projeto contempla:

- expansão da base local com mais **Normas Regulamentadoras (NRs)**  
- estruturação da base de conhecimento em **JSON**  
- separação dos módulos em arquivos Python  
- criação de interface web  
- melhoria da consulta por tema, contexto e fonte normativa  
- preparação para integração futura com mecanismos mais avançados de busca e IA  

### ⚠️ Observação Importante
O **SafeVoice SST** possui caráter **preliminar e orientativo**, não substituindo análise técnica em campo, interpretação formal da legislação ou validação por profissional habilitado.

## 🇺🇸 English

### 💙 About This Repository
This repository documents the evolution of the **SafeVoice SST** project.

It presents the development of a solution focused on **preliminary Occupational Safety guidance**, using voice interaction, automatic transcription, local knowledge lookup, and spoken responses.

The project was built with a focus on:
- accessibility;
- preliminary safety consultation;
- modular organization;
- application of **Software Engineering** and **Object-Oriented Programming** concepts.

### 🎓 Project Context
**SafeVoice SST** was developed from a practical challenge in the **GenAI & Data** learning track, within the bootcamp promoted by **Bradesco** in partnership with **DIO.me**.

The initial proposal was expanded with a more structured approach, connecting **speech transcription, text-to-speech, architectural organization, and practical application in Occupational Safety**.

### 📂 notebooks/
Initial functional version developed in **Google Colab**.

- browser-based audio recording  
- speech transcription with **Whisper**  
- local keyword-based knowledge lookup  
- guidance response in text  
- spoken response generation with **gTTS**  

### 📂 current logical structure
Version with an initial organization inspired by architectural principles.

- separation of responsibilities  
- class-based core services  
- division between transcription, knowledge base, text-to-speech, and flow control  
- structure prepared for future modularization into Python files  

### 🎯 Purpose
To demonstrate technical growth in building a solution applied to Occupational Safety:

**Voice Prototype → Structured Architecture → Foundation for an Intelligent Safety Assistant**

### 🤖 Next Step
The next evolution of the project includes:

- expanding the local knowledge base with more **Regulatory Standards (NRs)**  
- structuring the knowledge base in **JSON**  
- separating modules into Python files  
- creating a web interface  
- improving query logic by topic, context, and regulatory source  
- preparing the project for future integration with more advanced search and AI mechanisms  

### ⚠️ Important Note
**SafeVoice SST** has a **preliminary and guidance-oriented** purpose and does not replace on-site technical analysis, formal legal interpretation, or validation by a qualified professional.
